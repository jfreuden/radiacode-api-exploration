# Exploration of Radiacode Python API ☢️ 

I want to see what sort of fun stuff I can do with the Radiacode API. In particular, I am searching for interesting endpoints / outputs with two disparate goals in mind:

## 1. Scientifically useful ⚛️

Not necessarily research-mäßig, but anything generally useful. Think, "Would an undergrad physics lab be interested in this?"

## 2. Artistically useful 🎨

When I bought this device, I envisioned it playing some sort of aethereal music when it got interactions, based on the energy and count. I also thought it would be cool to have some sort of visual display that could show the energy and count in a way that was both informative and aesthetically pleasing.
